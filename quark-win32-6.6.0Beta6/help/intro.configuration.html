 1.3. Configuration of QuArK

  [ Prev - Up - Next ] 

In the configuration-window you can specify many things, to change the lookout and feel of QuArK. It is recommended that you study these settings, to learn what can and can't be done in QuArK.

Also, as QuArK is constantly going through changes, it would be wise to study the fly-over hints of the controls you're pointing at.

You can access the configuration-window by opening the  Edit  menu and then selecting the  Configuration  option when you aren't in the map/model editor, or using  Options  and then  Configuration  when you have opened a map or model.
