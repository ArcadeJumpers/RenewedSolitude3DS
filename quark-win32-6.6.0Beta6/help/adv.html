 3. Advanced customization

  [ Prev - Up - Next ] 

This section discusses the features of QuArK that can be customized without modifying the source code. This mainly includes:

Add-Ons: lists describing new entity types, new textures, etc. This is what you need to have QuArK support new Mods of the game or new games altogether.
Use of the QuArK Conversion Tools (see its section below) can help to create the needed addon .qrk files without having to copy files and write code.
  
Python: The language which much of the map-editor is written, and the major components of the Python portion of the editor, quarkpy and plugins.

See the section 'Map editing' for more information about how to use the map editor.
